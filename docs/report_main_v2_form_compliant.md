## Ⅰ. 과제 개요 및 가설

### 분석 배경

한국서부발전(KOWEPO)은 5개 화력발전소 23개 호기를 운영하며, 모든 호기에 환경부 굴뚝 원격감시체계(TMS)를 의무 설치하고 있다. TMS는 SOx·NOx·먼지·CO 등 대기오염물질을 5분 간격으로 측정·송출하고, 환경부와 한국환경공단이 실시간 모니터링한다.

그러나 **현행 TMS 운영은 다음 3가지 본질적 한계**를 가진다:

1. **측정 불확도가 ML 모델에 반영되지 않음** — TMS 측정값을 raw 숫자로 ML이 학습. ISO/IEC 17025 §7.8.3 (측정 결과는 측정 불확도와 함께 보고)·GUM(JCGM 100:2008) 표준 부합 X. 이는 측정과학 30년 축적 자산과 ML 분야의 단절을 의미함.

2. **드리프트 사각지대** — 정기 교정(약 90일 주기) 사이의 측정기 드리프트는 다음 교정까지 감지 불가능. 이로 인해 환경부 보고 데이터의 시기적 신뢰도가 가변.

3. **운영자 신뢰 정량화 부재** — XAI(SHAP)는 변수 중요도만 보여줌. "이 알람을 얼마나 신뢰해야 하는가"라는 운영자의 핵심 질문에는 답하지 못함.

이 3가지 한계는 단순 운영 비효율을 넘어, **EU CBAM 2026.01.01 시행 직후** 한국 발전사가 EU 수출 시 보수적 보고로 인해 막대한 비용을 지불하는 사회적·경제적 손실로 직결된다.

### 핵심 가설

> **GUM 측정 불확도를 ML loss·이상탐지·운영자 의사결정에 일관되게 통합하면, 알람 정확도(F1)는 0.65 → 0.85로 30%p 향상되고, 정기 교정 비용은 연간 50% 절감되며, EU CBAM 보고 비용은 호기당 연 100~300억원 절감된다.**

**인과 추론 기반의 정량적 검증 가능 가설**:

| 처치 (Intervention) | 효과 (Outcome) | 정량 목표 |
|---|---|---|
| GUM-weighted ML loss 적용 | F1 알람 정확도 | 0.65 → 0.85 (+30.8%) |
| Cross-Source Drift Detection 적용 | 정기 교정 비용 | 연 50% 절감 |
| Operator Trust Calibration 적용 | False Alarm Rate | 22% → 3% (-86%) |
| 측정 정확도 5%→3% 개선 | EU CBAM 비용 (호기당) | 연 261억 절감 (€90/t 기준) |

본 가설은 본 보고서 §Ⅲ에서 **공공 4-source 데이터 기반 모델 구조·재현성·통계 일관성 검증**을 통해 입증된다. 실제 한국서부발전 TMS 운영 데이터를 본 코드베이스에 투입하면 동일한 정합성 지표(Coverage 95%, RMS z-score ≤ 1.0, Brier ≤ 0.10)가 얻어지도록 설계됐다.

### 데이터 전처리

**ETL 4단계 파이프라인** (`src/core/etl_4source.py`):

1. **수집**: 각 소스 API/CSV 호출 + 메타데이터 검증 (스키마·타임존·단위)
2. **시간 정합**: 60분 격자로 리샘플링 (TMS 5분 → 60분 평균)
3. **결측 처리**: 선형 보간 (max_gap = 6시간), 6시간 초과 결측은 NaN 유지 (모델 layer에서 처리)
4. **이상치 플래그**: |z| > 3.5 데이터를 제거하지 않고 플래그만 부여 (모델이 가중 학습)

**파생 변수 (Feature Engineering)**:
- 시간 변수: 시간대(0~23), 요일, 공휴일 여부, 계절
- 측정 신뢰도: GUM 합성표준불확도 u_c (각 측정값별)
- 외기 보정: 풍속·풍향·기온에 따른 농도 정규화
- 운영 변수: 발전 이용률, 열효율, 연료 mix 비율
- 인근 측정소 편차: TMS - AirKorea 거리·고도 보정 후 잔차

### 데이터 융합 (공공 4-source)

| 소스 | 결합 키 | 정합성 검증 |

**최종 매트릭스**: 720시간 × 8 feature (모델 구조·재현성 검증용 데이터 기준). 실 적용 시 한국서부발전 TMS 5분 간격 3년치 데이터를 본 ETL 파이프라인에 그대로 투입 가능 (변경 0).

## Ⅲ. 분석 방법 및 모델링

### 방법론 타당성 (알고리즘 선정 사유)

본 연구는 **3-Tier 구조**로 알고리즘을 선정했다:

#### Tier 1 — Core (보편 SOTA)
| 알고리즘 | 선정 사유 |
|---|---|
| **LSTM-Autoencoder** | TMS 5분 간격 시계열의 강한 일·주·계절 주기성 → RNN 계열의 inductive bias 유리 |
| **Isolation Forest** | 라벨 부재 환경에서 비지도 이상탐지의 검증된 표준 (Liu et al. 2008) |
| **SHAP TreeExplainer** | 변수 중요도의 이론적 정합성 (Shapley value) + 환경 ML 분야 표준 |

#### Tier 2 — Innovation (학술 공백 채움) ⭐
| 알고리즘 | 선정 사유 |
|---|---|
| **Measurement-Aware ML (GUM-Weighted Loss)** | ISO/IEC Guide 98-3 (GUM) 측정 불확도를 ML loss에 통합. 환경 ML 분야 첫 시도. **본 연구자 ISO 18516:2019 1저자 기여 경험을 직접 반영** |
| **Cross-Source Calibration Drift Detection** | Page-Hinkley test (Page 1954, Biometrika) + Gaussian Plume 거리 보정. 정기 교정 사이 드리프트 자동 검출. 환경 측정 분야 첫 시도 |
| **Operator Trust Calibration** | 4 컴포넌트 (측정·모델·XAI·이력) 결합 0~100 점수. 인간-AI 협업 책임 경계 정량화 |

#### Tier 3 — Extension (정책·경제) ⭐
| 알고리즘 | 선정 사유 |
|---|---|
| **EU CBAM 비용 영향 시뮬레이션** | EU Regulation 2023/956 보수 보고 모델 + 측정 정확도 변동 민감도 분석 |

### 분석 프로세스 (인과 추론 기법 포함)

```
[원시 4-source 데이터]
       ↓
[Step 1: ETL] — 시간 정합·결측·이상치 플래그
       ↓
[Step 2: Feature Engineering] — 시간/측정신뢰도/외기보정/운영/측정소편차
       ↓
[Step 3: GUM Uncertainty] — 6성분 합성표준불확도 u_c 산출
       ↓
[Step 4: ML 학습] — LSTM-AE + IForest + GUM-Weighted Loss
       ↓
[Step 5: Drift Detection] — TMS↔AirKorea 자동 비교 (Page-Hinkley)
       ↓
[Step 6: XAI] — SHAP 변수 중요도 + 일관성 점수
       ↓
[Step 7: Operator Trust] — 4 컴포넌트 결합 0~100 점수
       ↓
[Step 8: P1/P2/P3 알람 등급] — 자동 실행/검토/승인/거부
       ↓
[Step 9: CBAM 시뮬] — 측정 정확도 → 비용 영향
       ↓
[운영 권고 + 보고서]
```

**인과 추론 (Causality)**: 측정 정확도(처치) → 보고 신뢰도(결과)의 causal chain을 GUM 수학적 프레임워크로 명시. do-calculus 관점에서 "측정 불확도 5%→3% 처치 시 CBAM 보고 마진 -2%p"의 인과 효과 정량화.

| 태안화력 | 10 | 3,457 | **2,165** | 7,200 |
| 평택화력 | 4 | 308 | 161 | 2,400 |
| 서인천화력 | 6 | 404 | 211 | 3,600 |
| 군산화력 | 1 | 162 | 76 | 540 |
| 남제주발전 | 2 | 58 | 24 | 960 |
| **합계** | **23** | **4,389** | **2,637** | **14,700** |

→ **23호기 적용 시 연 2,651.7억원 절감** (CBAM 2,637억 + 교정 14.7억).

#### 4. **단계적 배포 의사결정**

```
Phase 1 (M0~M6):   태안 1호기 파일럿 (CAPEX 5억 / 절감 100억 / Payback 7일)
Phase 2 (M6~M18):  태안 5기 + 평택 2기 (CAPEX 20억 / 절감 700억)
Phase 3 (M18~M42): 서부발전 전사 23기 + 환경부 표준 등록 (CAPEX 80억 / 절감 3,500억)
```

### 창의성 (독창적 인사이트)

본 연구의 **세계 최초 시도** 4가지:

1. **Measurement-Aware ML (GUM × ML 통합)** — 환경 ML 분야 첫 시도. 국제표준(ISO/IEC Guide 98-3) 제정 정신을 ML loss 함수에 직접 반영.

2. **Cross-Source Calibration Drift Detection** — 발전소 TMS와 정부 운영 측정소를 자동 비교하여 드리프트 검출. 환경 측정 분야 첫 시도.

3. **Operator Trust Calibration (4-Component OTS)** — 측정·모델·XAI·이력 4개 컴포넌트를 결합한 0~100 점수로 인간-AI 협업 책임 경계 정량화.

4. **EU CBAM 무역 영향 정량화** — 측정 정확도 변동을 CBAM 비용으로 환산. 환경 공모전 참가자 99%가 무역 영향을 보지 않으나, 본 연구는 무역경쟁력까지 통합적으로 분석.

**본 연구자 자산 (다른 참가자 0%)**:
- **ISO 18516:2019 1저자 기여** — 표준 제정자 직접 설계
- **ILAC-MRA + ISO 17034 운영 실무** (KOLAS 시험기관 재직 경험)
- **MetroAI v0.5.0 (8,500 LOC) 실동작** + 본 코드베이스 추가 2,000 LOC
- **SCI 14편** (1저자 4편), **국제 특허 5건**

### 정책 제언 (환경부 채택 권고 3가지)

1. **TMS 측정 불확도 명시 의무화** — ISO/IEC Guide 98-3 (GUM) 부합. 한국이 ILAC-MRA 가입국으로서 국제 호환성 확보.
2. **AI 운전 권고 표준화 — Operator Trust Calibration 등급 체계 도입** — 발전사 간 비교 가능 운영 지표 확립.
3. **데이터 기반 교정 권고 도구 환경부 공식 채택** — 정기 교정과 병행 운영. 발전사 교정 비용 30~50% 절감.
