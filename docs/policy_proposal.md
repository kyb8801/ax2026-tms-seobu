# 환경부 정책 제언 — TMS 운영 가이드라인 개정안 (1쪽)

> **수신**: 기후에너지환경부 / 한국환경산업기술원 (KEITI)
> **목적**: 본 솔루션의 정책 실현성 강화 — 평가 항목 직격
> **분량**: 보고서 별첨 1쪽

---

## 핵심 제언 3가지

### 1. TMS 측정 불확도 명시 의무화 (ISO/IEC Guide 98-3 부합)

**현행**: TMS 측정값을 단일 숫자로 보고. 측정 불확도는 부속서에만 기재.

**제안**: 모든 TMS 보고 데이터에 **합성표준불확도(u_c)와 확장불확도(U)를 명시**. 환경부 데이터베이스 스키마 개정.

**근거**:
- ISO/IEC 17025 §7.8.3 — 측정 결과는 불확도와 함께 보고해야 함
- 한국이 ILAC-MRA 가입국으로 국제 호환성 확보 필요
- EU CBAM도 향후 측정 불확도 보고 의무화 예상 (2027~)

**예상 효과**:
- 발전사 간 측정 정확도 비교 가능
- 환경부 정책 의사결정의 데이터 기반 정량화
- 국제 보고 (CBAM·UNFCCC) 직접 활용

### 2. AI 운전 권고 시스템 표준화 — Operator Trust Calibration 도입

**현행**: 발전사별로 자체 알람 기준 운영. 일관성 부재.

**제안**: 본 보고서 §7의 **0~100 운영자 신뢰 점수(OTS) 등급 체계를 표준화**. P1/P2/P3 자동 알람 등급을 환경부 공통 가이드라인에 반영.

**구성 원칙**:
- 측정 신뢰도 (GUM)
- 모델 신뢰도 (예측 분산)
- XAI 일관성 (SHAP 안정성)
- 과거 적중률

**예상 효과**:
- 발전사 간 비교 가능 운영 지표 확립
- 운영자 책임 범위 명확화 (P1 자동 / P2 검토 / P3 승인)
- 환경부 감독·심사 효율 ↑

### 3. 측정기 드리프트 데이터 기반 교정 권고 (현행 정기 교정 보완)

**현행**: 모든 발전사 90일 정기 교정. 사이 드리프트는 감지 X.

**제안**: 본 보고서 §6의 **Cross-Source Calibration Drift Detection을 환경부 공식 권고 도구로 채택**. 정기 교정과 병행 운영.

**근거**:
- 인근 에어코리아 측정소 데이터로 자동 비교 → 추가 인프라 불요
- 현행 정기 교정은 그대로 유지 (안전 마진)
- 데이터 기반 권고는 *조기 경보* 역할

**예상 효과**:
- 측정 정확도 항상 유지 (정기 교정 사이 사각지대 해소)
- 발전사 교정 비용 30-50% 절감
- ESG·CBAM 보고 신뢰도 ↑

## 시범 운영 권고

**대상**: 한국서부발전 5개 호기 (태안·평택·서인천·군산·남제주)
**기간**: 2026.07 ~ 2027.06 (1년)
**평가지표**: §9 KPI 정의서 참조

**1단계 (2026 하반기)**: 측정 불확도 보고 시범 + OTS 도입 → 운영자 피드백 수집
**2단계 (2027 상반기)**: Drift Detection 시범 + CBAM 보고 정확도 비교
**3단계 (2027 하반기)**: 효과 평가 → 5대 발전사 (한수원·동서·서부·남부·중부) 확대

## 관련 부합 표준·법령

| 표준·법령 | 본 제언과의 관계 |
|---|---|
| ISO/IEC 17025:2017 | 측정 불확도 보고 의무 (§7.8.3) |
| ISO/IEC Guide 98-3:2008 (GUM) | 합성표준불확도 산출 방법 |
| ISO 17034:2016 | 인증표준물질 (교정용 기준 가스) |
| ISO 18516:2019 | (본 연구자 1저자 기여) 측정 능력 평가 |
| 대기환경보전법 시행규칙 별표 18 | TMS 운영·관리 기준 |
| 환경부 굴뚝 원격감시체계 운영 가이드라인 | 본 제언으로 개정 권고 |
| EU CBAM Regulation 2023/956 | 2026.1.1 시행, 측정 보고 정확도 영향 |

## 결언

본 정책 제언은 단순 발전사 비용 절감을 넘어, **한국이 국제 환경 측정 표준의 모범 사례를 만드는 기회**다. ISO/IEC 17025·GUM 표준 부합 + EU CBAM 호환 + 발전사 간 비교 가능 시스템은 향후 ISO TC 146 (대기 측정) 및 ISO TC 207 (환경 관리) 국제표준 제안 기반이 될 수 있다.
