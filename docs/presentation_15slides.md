## 슬라이드 6 — Innovation A: Measurement-aware ML ⭐

```
┌─────────────────────────────────────────────────────────────┐
│   ⭐ Innovation A: Measurement-aware ML                       │
│                                                             │
│   기존 ML:                                                   │
│   ┌────────┬────────┬────────┐                              │
│   │측정 1.0│측정 5.0│측정 0.5│ ←  모두 동등 가중             │
│   │ ±0.5  │  ±5   │ ±0.5  │                              │
│   └────────┴────────┴────────┘                              │
│                                                             │
│   본 솔루션 (GUM-weighted Loss):                              │
│   ┌────────┬────────┬────────┐                              │
│   │  4.0   │  0.04  │   4.0  │ ← 신뢰도 ↑ → 가중 ↑          │
│   └────────┴────────┴────────┘                              │
│                                                             │
│   L = Σ (1/u_c²) × (y - ŷ)²                                  │
│                                                             │
│   ✅ ISO/IEC Guide 98-3 (GUM) 6성분 합성표준불확도            │
│   ✅ Heteroscedastic Gaussian NLL                            │
│   ✅ MC Uncertainty Propagation → 출력 신뢰구간 자동           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**음성 (60초)**: "Innovation A — Measurement-aware ML. 기존 ML은 모든 측정 데이터를 동등하게 학습합니다. 본 솔루션은 GUM 합성표준불확도의 역수 제곱을 가중치로 사용해, 정밀한 측정에 더 큰 학습 영향을 부여합니다. ISO 17025 6개 불확도 성분을 직접 구현했고, Heteroscedastic Gaussian NLL과 Monte Carlo Uncertainty Propagation으로 출력에도 신뢰구간이 자동으로 따라옵니다. 코드는 GitHub에 공개되어 있고, 평가위원께서 직접 실행 가능합니다."
